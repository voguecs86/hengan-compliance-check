💻 C2PA签名命令行工具使用指南

Pro版签名功能基于开源 c2pa-python 库（Apache 2.0）。以下命令可直接在你的电脑上运行：

安装依赖pip install c2pa-python cryptography
签名命令：
# AI生成内容签名
python c2pa_signer.py sign input.jpg output_signed.jpg --ai-generated

# AI增强内容签名
python c2pa_signer.py sign input.jpg output_signed.jpg --ai-enhanced

# 使用demo证书签名（测试验证）
python c2pa_signer.py sign input.jpg output_signed.jpg --ai-generated --demo

# 验证已签名图片
python c2pa_signer.py verify signed_image.jpg

# 批量签名（整个目录）
python c2pa_signer.py batch ./images ./signed_output --ai-generated

签名后的manifest包含以下核心声明：

c2pa.actions.v2 -- AI来源声明
cawg.training-mining -- 训练数据声明
重要说明:
  • Demo模式使用C2PA官方测试证书，验证状态为 Trusted
  • 生产环境需替换为正式C2PA信任锚证书（从C2PA授权CA获取）
  • 脚本与证书文件已包含在本仓库中: GitHub仓库
